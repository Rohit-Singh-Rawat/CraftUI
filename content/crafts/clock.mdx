## Inspiration

I saw a post of a clock design and thought of building a clockâ€”why not? This component brings that inspiration to life with a fully customizable analog clock that supports multiple timezones, custom styling, and beautiful variants.

## Features

- **Multiple variants** - Choose from mesh background or clean default styles
- **Timezone support** - Display time from any IANA timezone (e.g., 'America/New_York', 'Asia/Tokyo')
- **Custom time** - Set a static custom time for demonstrations or testing
- **Highly customizable** - Control tick marks, hand colors, numbers, borders, and more
- **Digital overlay** - Optional digital time and weekday display with flexible positioning
- **Responsive sizing** - Scales proportionally from a base 320px design
- **Smooth animations** - Real-time updates with configurable refresh rates

## Usage

<CodeBlock
	code={`import { Clock } from '@/craft/components/clock';

// Basic clock with mesh variant
<Clock variant="mesh" size={320} />

// Clock with timezone
<Clock 
  variant="default" 
  timezone="America/New_York"
  showNumbers="cardinal"
/>

// Fully customized clock
<Clock
  variant="mesh"
  size={200}
  showNumbers="all"
  rounded={1}
  borderWidth={2}
  borderColor="rgba(255,255,255,0.8)"
  tickHourColor="rgba(255,255,255,0.8)"
  handHourColor="#fff"
  handSecondColor="#ff0000"
  showDigital={{
    position: "center",
    uppercase: true,
    showTimezone: true
  }}
/>`}
	language='tsx'
/>

## Design Notes

The clock uses SVG for crisp rendering at any size. All measurements are based on a 320px design and scale proportionally. The component supports both analog and digital time displays, making it perfect for dashboards, world clocks, or any time-sensitive UI.

The mesh variant features a beautiful gradient background with grain texture, while the default variant uses your theme's background color for seamless integration.
