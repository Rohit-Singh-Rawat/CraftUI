# Fluid Asset Preview

Morphing asset card with a split layout, rolling dropdown, and animated CTA so files feel alive instead of static.

## Craft Story

This component was inspired by a design shared on X (formerly Twitter) by [@ayushsoni_io](https://x.com/ayushsoni_io/status/1985645474771321194). The original design featured a clean, fluid asset preview card with an elegant morphing action menu.

## How to Use

<CodeBlock
	code={`import { FluidAssetPreview } from '@/craft/components/fluid-asset-preview';

<FluidAssetPreview
  fileName="document.pdf"
  fileDate="January 15, 2024"
  thumbnailUrl="/path/to/thumbnail.png"
  fileType="pdf"
  onLinkClick={() => console.log('Link clicked')}
/>`}
	language='tsx'
/>
