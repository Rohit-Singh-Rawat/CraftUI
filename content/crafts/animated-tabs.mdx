# Animated Tabs

Animated pill navigation paired with a blur-fade content stage so you can tell long-form product stories without switching sections.

## Craft Story

I saw a hero section on Doppler.com that stacked headline, stats, and integrations onto one cinematic surface, so I re-created the interaction with a springy pill, shared `layoutId`, blur-in copy, and a generous content stage tuned for narrative-heavy launches.

## How to Use

<CodeBlock
	code={`import AnimatedTabs from '@/craft/components/animated-tabs';

const tabs = [
{
label: 'Security',
title: 'Secure secrets',
content: (

<div className="space-y-3">
<p className="text-base text-muted-foreground">
Narrative copy with compliance callouts, breach data, and a CTA stack.
</p>
<div className="grid gap-2 md:grid-cols-3">
<div className="rounded-xl border border-border bg-background/80 p-3">
<p className="text-xs uppercase tracking-wide text-muted-foreground">Uptime</p>
<p className="text-2xl font-serif">99.99%</p>
</div>
{/* ...more stat cards */}
</div>
</div>
),
},
{
label: 'Integrations',
title: 'Central hub',
content: <p>Logos, badges, and webhook copy go here.</p>,
},
];

export function StoryTabs() {
  return (
    <AnimatedTabs
      tabs={tabs}
      containerClassName="mx-auto"
      tabClassName="font-serif text-sm md:text-base"
      activeTabClassName="bg-primary/5 border border-border/40"
      contentClassName="bg-muted/40 backdrop-blur-xl"
    />
  );
}`}
	language='tsx'
/>
